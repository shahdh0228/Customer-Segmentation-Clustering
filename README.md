# 🛍️ Customer Segmentation (Clustering) - Task 2

## Objective
Segment mall customers based on **Annual Income** and **Spending Score** using unsupervised learning.

## Methods
- Data scaling (StandardScaler)
- K-Means clustering (optimal K via Elbow Method)
- DBSCAN clustering (bonus) + outlier detection

## Key Outputs
- Elbow plot to choose K
- Cluster visualization plots (K-Means & DBSCAN)
- Cluster-wise average income/spending analysis

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Jupyter Notebook

## Structure
- `data/` → dataset
- `notebooks/` → notebook
